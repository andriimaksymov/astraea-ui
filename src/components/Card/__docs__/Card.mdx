# Card

A card component for displaying content with a border, padding, and optional elevation.

#### Example

<Canvas of={Card.Default} />

## Usage

```ts
import { Card } from "awesome-ui-library";

const Example = ({ children }) => {
  return (
      <Card>
        {children}
      </Card>
  );
};

export default Example;
```

## Props

### `hasBorder` (boolean)

Specifies whether the card should have a border. Default: `true`.

### `borderColor` (string)

The color of the card's border. Accepts any valid CSS color value.

### `className` (string)

Additional CSS classes to apply to the card element.

### `children` (ReactNode)

The content to render inside the card.

### `padding` (string)

The padding size of the card. Options: `none`, `small`, `medium`, `large`. Default: `medium`.

### `elevation` (number)

The elevation level of the card. This determines the shadow effect. Options: `0`, `1`, `2`, `3`, `4`, `5`. Default: `1`.
